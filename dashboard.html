Sprite Customization Guide

    

      Sprite files: ~/.tetora/media/sprites/
    

    

      

        
Image Format

        

          
Size128 x 64 px

          
Cell16 x 32 px (8 cols x 2 rows = 16 frames)

          
FormatPNG, transparent background, no anti-aliasing

          
Style16-bit chibi pixel art (2.5 head proportion)

        

      

      

        
Grid Layout


128px
├─16─┤
┌──┬──┬──┬──┬──┬──┬──┬──┐
│00│01│02│03│04│05│06│07│ Row 0: front-facing (walk + idle/action)
├──┼──┼──┼──┼──┼──┼──┼──┤
│08│09│10│11│12│13│14│15│ Row 1: back/side-facing (walk + alt actions)
└──┴──┴──┴──┴──┴──┴──┴──┘

      

      

        
Cell Rules

        
    
          
  • Character centered horizontally in each 16px cell
    • 
          
  • Feet aligned to bottom of each 32px cell
    • 
          
  • Consistent position across all frames (no jittering)
    • 
          
  • No part extends beyond cell boundary
    • 
        

      

      

        
config.json


{
  "cellWidth": 16, "cellHeight": 32,
  "states": [
    { "name": "idle",  "row": 0, "frames": 4 },
    { "name": "work",  "row": 0, "frames": 4 },
    { "name": "think", "row": 1, "frames": 4 },
    ...
  ],
  "agents": {
    "ruri": { "sheet": "ruri.png" }
  }
}

        

          row = which row (0=top, 1=bottom)  |  frames = how many cols to cycle
        

      

      

        
Animation

        
    
          
  • 200ms per frame (5 FPS), auto-looping
    • 
          
  • Walk cycle: left-step → stand → right-step → stand
    • 
          
  • Agents walk between rooms based on task status
    • 
          
  • idle → lounge, work/think → desk, talk/review → meeting room
    • 
        

      

      

        
AI Prompt — Gemini

        


Generate a pixel art character sprite sheet for a retro RPG game.

Character: [DESCRIBE CHARACTER HERE]

Requirements:
- Image size: exactly 128 x 64 pixels
- Grid layout: 8 columns x 2 rows, each cell is 16x32 pixels
- Style: 16-bit SNES/RPG Maker chibi, 2-3 head tall
- Background: fully transparent (PNG)
- No anti-aliasing, hard pixel edges only
- Each cell contains exactly ONE pose, centered, feet at bottom

Row 0 (top): front-facing — 4 walk cycle + 4 idle/action
Row 1 (bottom): back-facing — 4 walk cycle + 4 side-view

Consistent palette across all 16 frames. No borders or grid lines.

          
        

      

      

        
AI Prompt — ChatGPT

        


Create a single pixel art sprite sheet PNG, exactly 128x64 pixels.
Grid: 8 columns x 2 rows. Each cell: 16px wide x 32px tall.

Character: [DESCRIBE CHARACTER HERE]

Style: 16-bit retro RPG chibi (RPG Maker / FF6), 2.5-head proportion, hard pixel edges, no smoothing, transparent background, limited palette (12-16 colors).

Row 1 (top 32px): front-facing walk cycle (4 frames) + idle/action (4 frames)
Row 2 (bottom 32px): back-facing walk cycle (4 frames) + alt actions (4 frames)

CRITICAL: Each character centered in 16x32 cell. Feet at bottom. No grid lines. Exactly 128x64 pixels.

          
        

      

      

        
Checklist

        
    
          
  1. Image is exactly 128x64 px
    2. 
          
  2. Each 16x32 cell has one pose only
    3. 
          
  3. Feet aligned, character centered, no jitter
    4. 
          
  4. Transparent background, no white edges
    5. 
          
  5. Recognizable at actual 16px size
    6. 
          
  6. Same palette across all frames
    7.
